## ✨ 功能特色

### 📱 響應式設計
- **桌面版**：完整的表格檢視，適合大螢幕瀏覽
- **手機版**：卡片式設計，避免四欄擠在一起，閱讀更輕鬆

### 🗺️ 詳細地址資訊
- 點擊「▼ 詳細地址」可展開/收合地址
- 地址可直接點擊開啟 Google Maps 導航
- 包含捷運站出口與步行時間

### 🔗 直接預約
- 預約制店家提供直接連結
- 一鍵開啟官網預約系統
- LINE ID 可快速複製搜尋

### 🎨 視覺化標籤
- 🟢 **綠色標籤** - 免預約店家（Walk-in）
- 🔴 **紅色標籤** - 全預約制
- 🟡 **黃色標籤** - 雙軌制（推薦預約）
- 🔵 **藍色標籤** - 推薦預約

## 🚀 使用方式

### 線上瀏覽
直接訪問 Google Sites：
```
https://sites.google.com/view/find-mattress
```

### 本地瀏覽
1. 下載 `index.html`
2. 用瀏覽器開啟即可

## 💻 技術架構

- **框架**：純 HTML + Tailwind CSS
- **字體**：Google Fonts (Noto Sans TC)
- **響應式**：CSS Media Queries + JavaScript 偵測
- **相容性**：支援桌面、平板、手機瀏覽器

### 手機偵測機制
使用多重偵測方式確保在各種環境下都能正確切換版面：
- CSS Media Queries (寬度、裝置寬度、觸控能力)
- JavaScript User Agent 偵測
- 視窗大小即時監聽

## ⚠️ 注意事項

1. **營業時間可能變動**，建議前往前先致電確認
2. **預約制店家務必提前預約**，避免白跑一趟
3. **部分品牌有多個據點**，請確認正確的門市地址
4. **Emma 品牌**：免預約僅限竣樂家居門市，設計公司據點需預約
